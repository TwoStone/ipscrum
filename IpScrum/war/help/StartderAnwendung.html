Anwendungsstart


IP-Scrum ist eine Webanwendung und wird aus diesem Grund direkt vom Internetbrowser aus aufgerufen. Eine Installation oder sonstige Änderungen an Ihrem System sind nicht notwendig.


Unterstützt werden folgende Browser:

  • Google Chrome (alle Versionen)
  • Apple Safari (ab Version 4)
  • Mozilla Firefox (ab Version 3.6)
  • Microsoft Internet Explorer (ab Version 9)


Um eine optimale Darstellung der Anwendung zu gewährleisten wird die Nutzung WebKit-basierter Browser (Chrome oder Safari) empfohlen.
